What Is It?

Modified rubrics are scaffolded assessment tools that provide additional structure and support while maintaining the same learning standards as general education rubrics. Unlike accommodations that change how a student accesses content, modified rubrics provide temporary scaffolds that fade over time as students build skills. This is a Tier 2 intervention because it targets specific students who need additional support to demonstrate their learning.

Research Base: Scaffolded assessments have an effect size of 0.82 (Hattie), making them one of the most powerful interventions for student achievement. The key is that scaffolds are temporary - they're designed to build independence, not create dependency. Modified rubrics help students understand expectations while building the skills to meet those expectations independently.
Key Principle: Modified rubrics are NOT "easier" or "less rigorous." They provide the same end goal with additional structure, examples, and checkpoints. The goal is for students to eventually use the general education rubric independently.

What Quality Implementation Looks Like

‚úÖ In the Classroom (What TO Do):

  • Modified rubric maintains the same learning standards as general rubric
  • Scaffolds are explicit and visible (students know what extra support they're getting)
  • Scaffolds are temporary - plan includes fading support over time
  • Students receive instruction on how to use the modified rubric
  • Modifications are based on data about specific needs, not assumptions
  • Modified rubrics are reviewed with students before, during, and after the assignment
  • Progress toward using standard rubric is monitored and celebrated

‚ùå Red Flags (What NOT to Do):

  • Modified rubric has lower standards or different learning goals
  • All struggling students get same modification (not individualized to need)
  • Scaffolds become permanent - student never transitions to standard rubric
  • Student doesn't understand why or how to use modified version
  • Modified rubric stigmatizes student or makes them feel "less than"
  • No plan for fading support over time
  • Modification becomes a crutch rather than a bridge to independence

Types of Rubric Modifications

Modification Type What It Provides Best For Example
Scaffolded Criteria Breaks each criterion into smaller, more specific steps Students who understand the concept but struggle with complex tasks Instead of "Write a well-organized paragraph," break into:
- Topic sentence present
- 3+ supporting details
- Details connect to topic
- Conclusion sentence
Exemplars Embedded Includes examples directly in the rubric for each level Visual learners; students needing concrete examples Level 4: "Strong evidence" might show:
"The author uses the metaphor 'life is a journey' to suggest..."

"The text says 'life is a journey' which means..."

"According to the text, _____ . This shows that _____."

‚òê 3 body paragraphs (6 pts)

Week 5-6: Add "Conventions" (full rubric)

Step-by-Step Implementation

Phase 1: Identify Need (Before Creating Modified Rubric)

Step 1: Analyze Student Performance Data

Don't modify without data! Understand WHY the student is struggling.

Example - Writing Assignment: Student scores consistently low on writing rubric. Look at patterns: - Does student struggle with ALL criteria equally? ‚Üí May need writing intervention (Tier 3) - Does student score low on "organization" but well on "ideas"? ‚Üí Scaffolded rubric for organization - Does student write very little or seem overwhelmed? ‚Üí Checklist format to break down task - Is student ELL or struggle with academic language? ‚Üí Sentence frames Decision: Based on pattern, student needs scaffolded support for "organization" criterion specifically.

Step 2: Determine Which Modification Type Fits the Need

Decision Tree:

  • If student understands task but struggles with complexity: ‚Üí Scaffolded Criteria
  • If student doesn't understand what quality looks like: ‚Üí Exemplars Embedded
  • If student has language barriers: ‚Üí Sentence Frames
  • If student has executive functioning/organization issues: ‚Üí Checklist Format
  • If student is overwhelmed by multiple criteria: ‚Üí Reduced Complexity

Phase 2: Create the Modified Rubric

Step 3: Build the Scaffold While Maintaining Standards

Original Rubric (Science Lab Report - Organization Criterion): Level 4: Lab report is exceptionally well-organized with clear sections Level 3: Lab report is organized with identifiable sections Level 2: Lab report has some organization but sections unclear Level 1: Lab report lacks organization Modified Rubric (Scaffolded Criteria): Level 4 - All sections present and complete (4 pts): ‚òê Question clearly stated ‚òê Hypothesis with "if...then..." format ‚òê Materials list (complete) ‚òê Procedure (numbered steps) ‚òê Data table (labeled) ‚òê Conclusion answers question Level 3 - Most sections present (3 pts): 5 of 6 sections above completed Level 2 - Some sections present (2 pts): 3-4 of 6 sections above completed Level 1 - Few sections present (1 pt): 0-2 sections completed Note: Same standard (organization), but scaffolded into explicit checklist

Step 4: Add Visual Supports

  • Color-coding: Highlight different sections or criteria in different colors
  • Icons: Add visual markers (‚úì for checklist items, ‚≠ê for highest level)
  • Progress bars: Show how close student is to each level
  • Examples: Include side-by-side comparison of Level 2 vs Level 4 work

Phase 3: Teach Students to Use Modified Rubric

Step 5: Explicit Instruction on Rubric Use

Don't just hand it out - TEACH students how to use it!

Mini-Lesson on Using Modified Rubric (15 minutes):

1. Explain the Purpose (3 min):

"This rubric is designed to help you understand exactly what I'm looking for in your lab report. It breaks down 'organization' into specific things you need to include. Let's look at it together."

2. Walk Through Each Section (5 min):

"For Level 4, you need all six sections. Let me show you what each one means..."
[Go through each checklist item with example]

3. Practice with Sample Work (5 min):

"Here's a sample lab report. Let's use the rubric to score it together. First, check if there's a clear question..."
[Model using the checklist to evaluate work]

4. Self-Assessment (2 min):

"Before you turn in your work, you'll use this rubric to score yourself. This helps you check if you've included everything."

Phase 4: Fade the Scaffold Over Time

Step 6: Plan for Gradual Release

Modified rubrics are a bridge, not a permanent accommodation!

Fading Plan Example (8-Week Timeline): Weeks 1-2: Full Scaffold - Student uses modified rubric with all scaffolds - Teacher provides explicit instruction on each criterion - Student self-assesses using checklist before submitting ‚Üí Goal: Student scores Level 3 or higher using scaffold Weeks 3-4: Partial Scaffold - Remove some checklist items (keep the hardest ones) - Student fills in removed items from memory - Teacher provides feedback on use of rubric ‚Üí Goal: Student can identify 80% of criteria independently Weeks 5-6: Minimal Scaffold - Provide only rubric categories (no checklist details) - Student creates own checklist from categories - Teacher reviews student-created checklist for accuracy ‚Üí Goal: Student demonstrates understanding of expectations Weeks 7-8: Standard Rubric - Student uses general education rubric - Teacher monitors for continued success - Scaffold available as reference if needed ‚Üí Goal: Student independently uses standard rubric

Step 7: Monitor Progress Toward Independence

Progress Monitoring Questions: - Is student's score improving over time? - Can student explain what each rubric level means? - Does student use rubric for self-assessment before submitting work? - Can student identify what level their work meets? - Is student ready for less scaffolding? Data to Track: Week 1: Used full scaffold ‚Üí scored 3/4 Week 3: Used full scaffold ‚Üí scored 4/4 (ready to fade!) Week 5: Used partial scaffold ‚Üí scored 3/4 Week 7: Used standard rubric ‚Üí scored 3/4 (SUCCESS!)

Common Pitfalls & Solutions

‚ùå Pitfall: Scaffold Becomes Permanent

Problem: Student continues using modified rubric all year without moving toward standard rubric

‚úÖ Solution:

  • Build fading plan from Day 1
  • Set specific timeline (e.g., 6-8 weeks)
  • Monitor progress weekly
  • Celebrate reduction of scaffolds as growth
  • If student isn't ready to fade, may need Tier 3 support

‚ùå Pitfall: Modified Rubric Lowers Standards

Problem: Student is assessed on easier content or fewer skills

‚úÖ Solution:

  • Ensure modified rubric addresses SAME standards
  • Scaffold is about support, not reducing expectations
  • Level 4 on modified rubric = Level 4 on standard rubric
  • Only difference is how clearly expectations are broken down

‚ùå Pitfall: Student Doesn't Understand How to Use It

Problem: Rubric handed out but not explained; student ignores it

‚úÖ Solution:

  • Provide explicit instruction on rubric use
  • Model using rubric to evaluate sample work
  • Practice self-assessment together
  • Check in during work time: "Show me where you are on the rubric"

‚ùå Pitfall: One-Size-Fits-All Modification

Problem: All struggling students get same modified rubric regardless of need

‚úÖ Solution:

  • Use data to identify SPECIFIC struggle for each student
  • Student A might need checklist format
  • Student B might need sentence frames
  • Student C might need exemplars
  • Modification should match the specific barrier

Subject-Specific Examples

Math: Problem-Solving Rubric

Standard Rubric: "Shows complete work"

Modified Scaffold:

  • ‚òê Read and underline key information
  • ‚òê Write what you're solving for
  • ‚òê Show equation/setup
  • ‚òê Show all calculation steps
  • ‚òê Write answer with units
  • ‚òê Check if answer makes sense

ELA: Text Analysis Essay

Standard Rubric: "Provides strong textual evidence"

Modified with Sentence Frames:

  • The author states, "____" (pg. ___)
  • This evidence shows that ____
  • Another example is "____" (pg. ___)
  • This demonstrates ____ because ____

Science: Lab Report

Standard Rubric: "Draws valid conclusions"

Modified Checklist:

  • ‚òê Restate the question
  • ‚òê Answer: Was hypothesis supported?
  • ‚òê Cite data from results table
  • ‚òê Explain what data means
  • ‚òê State sources of error

Social Studies: DBQ Essay

Standard Rubric: "Analyzes multiple documents"

Modified with Exemplars:

Level 4 Example: "Document A shows..., while Document B suggests... These sources together reveal..."

Level 3 Example: "Document A shows... Document B shows... Both sources discuss..."

Documentation & Communication

Documenting Modified Rubric Use:

  • Why: Track what modification was used and student progress toward standard rubric
  • What to document:
    • Date scaffold introduced
    • Type of modification used
    • Student scores over time
    • When scaffold was faded
    • Student's success with standard rubric
  • Where: Student intervention log, gradebook notes, or MTSS documentation

Communicating with Parents:

Sample Email/Note:

"Hello [Parent], I wanted to share a support strategy I'm using with [Student] for writing assignments. I've created a modified rubric that breaks down the writing expectations into more specific, concrete steps. This helps [Student] understand exactly what's expected and check their own work before submitting. This is a temporary scaffold - as [Student] becomes more confident with the writing process, we'll gradually remove the extra support. I'm seeing great progress already! The modified rubric doesn't change the learning standards - [Student] is working toward the same goals as the rest of the class, just with additional structure to support success. Please let me know if you have any questions!"

Quick Start Guide

Create Your First Modified Rubric:

Step 1: Identify Student (10 min)
  • ‚òê Look at recent assessment data
  • ‚òê Which student consistently scores low on same rubric criterion?
  • ‚òê What specifically are they struggling with?
Step 2: Choose Modification Type (5 min)
  • ‚òê Review types of modifications table above
  • ‚òê Select the type that matches student's specific struggle
Step 3: Create Modified Version (15 min)
  • ‚òê Take your existing rubric
  • ‚òê Add appropriate scaffold (checklist, frames, exemplars, etc.)
  • ‚òê Ensure standards remain the same
Step 4: Teach Student to Use It (15 min)
  • ‚òê Meet with student to explain rubric
  • ‚òê Practice using it on sample work
  • ‚òê Have student try self-assessment
Step 5: Monitor & Fade (Ongoing)
  • ‚òê Track student scores over time
  • ‚òê After 3-4 successful uses, begin fading scaffold
  • ‚òê Transition to standard rubric within 6-8 weeks
