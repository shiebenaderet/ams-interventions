What It Is

Students can show understanding through different formats (written, oral, visual, performance) while still meeting the same learning target.

What Quality Implementation Looks Like

‚úÖ In the Classroom (What TO Do):

  • Learning target stays the same (non-negotiable)
  • 2-4 format options provided
  • Clear rubric shows expectations for each format
  • Instruction includes models for each format
  • Choice builds student agency without lowering rigor
  • Students learn to match format to their strengths

‚ùå Red Flags (What NOT to Do):

  • "Do whatever you want" with no structure
  • Learning target changes based on format (different rigor)
  • Options include only one accessible format (others too hard)
  • No models or instruction for alternative formats
  • Students always pick easiest option
  • Teacher can't assess learning because format obscures understanding

Implementation Steps

Step 1: Design the Assignment with Choice

  • Start with the Learning Target (non-negotiable)
    What must students know/be able to do?
  • Brainstorm Format Options
    Written, oral, visual, kinesthetic, digital
    Choose 3-4 options that all allow students to meet the target
  • Create Common Rubric
    Content criteria (same for all formats)
    Format-specific criteria (unique to each)
Example Rubric: Criteria | All Formats | Format-Specific ---------|-------------|---------------- Content | ‚Ä¢ Shows clear understanding of [concept] | Essay: Writing conventions, paragraph structure | ‚Ä¢ Includes 2+ pieces of evidence | Visual: Organization, labels, design supports content | ‚Ä¢ Explains reasoning | Oral: Eye contact, volume, pace

Step 2: Teach Students About Choice

Mini-Lesson on Making Strategic Choices (10 min): Teacher: "When I give you choice, how do you decide? Some students always pick the same thing. That's okay, but I want you to think strategically: ‚Ä¢ Which format helps YOU show what you know best? ‚Ä¢ Which format challenges you to grow? ‚Ä¢ Which format matches this content? Let me show you..." [Model thinking through choice for a sample assignment]

Step 3: Provide Models

For each format option:

  • Show example from past student (with permission) OR
  • Create your own example OR
  • Show example from another source

When showing models:

  • Point out what makes it successful
  • Highlight how it meets the learning target
  • Show different quality levels if possible (to calibrate expectations)

Step 4: Build in Checkpoints

Don't wait until final product to see student work:

Checkpoint All Students Purpose
Proposal (Day 1) "I'm choosing [format] because..." Ensure strategic choice
Draft/Outline (Day 3) Rough draft or plan Catch misunderstandings early
Peer Review (Day 5) Partner feedback using rubric Quality check before submission
Final (Day 7) Submit final product Assessment

Step 5: Reflect on Choice

After assignment, students reflect:

  • "Which format did you choose and why?"
  • "Did this format help you show your learning? Why or why not?"
  • "Next time, would you choose the same or try something different?"

This builds metacognition about their learning preferences.

Scaffolding Choice

Level 1: Teacher-Directed

(Start here)

  • Teacher chooses format for all students
  • All students practice same format with support

Level 2: Structured Choice

(What we described above)

  • Teacher provides 2-4 format options
  • All options meet same learning target
  • Clear rubric for each

Level 3: Open Choice

(Advanced)

  • "Show your understanding in any format you choose"
  • Student proposes format
  • Teacher approves if it allows assessment of learning target
Most students need Level 2. Move to Level 3 only when students demonstrate strategic choice-making.

Common Pitfalls & Solutions

‚ùå Pitfall: Letting students always pick "easy" option

‚úÖ Solution: Require students to try each format at least once per quarter

‚ùå Pitfall: Options are not equally rigorous

‚úÖ Solution: Use common content rubric; adjust only format-specific criteria

‚ùå Pitfall: Too many options (choice paralysis)

‚úÖ Solution: Limit to 3-4 clear options

‚ùå Pitfall: Students don't know what formats "look like"

‚úÖ Solution: Always provide models

‚ùå Pitfall: Choice becomes more important than learning

‚úÖ Solution: Always center the learning target, not the product
