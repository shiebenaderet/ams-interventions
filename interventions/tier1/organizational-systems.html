What It Is

Consistent structures that help all students track assignments, materials, and deadlines.

What Quality Implementation Looks Like

‚úÖ In the Classroom (What TO Do):

  • Every student has the same organizational tool (binder system, notebook, digital folder structure)
  • Teacher uses the same system and models it daily
  • First 5 minutes of class: students get organized (materials out, check Canvas, update agenda)
  • Visual systems posted (what goes in each binder section, how to name Canvas files)
  • Students can find materials from 2 months ago in under 60 seconds

‚ùå Red Flags (What NOT to Do):

  • "I told them to get organized" but no system taught
  • Different organizational systems for different classes
  • Teacher doesn't use the system they're asking students to use
  • No time built into class for organization
  • Students with missing materials punished rather than re-taught

Implementation Steps

Week 1-2: Launch

  • Explicitly teach the system (don't assume they know)
  • Practice the system with low-stakes activities
  • Check every student's system individually
  • Give specific feedback: "I see you put today's notes in the right section. Now add the date at the top."

Week 3-6: Practice

  • Start every class with 3-minute organization routine
  • Use visual timer
  • Spot check 5 random students per day
  • Celebrate students doing it right

Week 7+: Maintenance

  • Weekly "binder checks" or Canvas audits
  • Students self-assess their organization
  • Re-teach as needed (especially after breaks)
  • Connect to progress monitoring: "When you organized better, your grades improved"

Collaboration with Other Teachers

  • Agree on same binder sections across classes (if using binders)
  • Use same Canvas naming conventions
  • Shared expectations for what "organized" means
